## 機器人指令

- 加入獎項至 Google Sheets

  `/add [item1] [item2] [item3] ...`
- 登記
  
  `/want [who] [item1] [item2] [item3] ...`
- 各個 item 報名情況
  
  `/list`
- 抽獎
  
  `/lottery [item1] [count]`
- 指令說明
  
  `/help`
